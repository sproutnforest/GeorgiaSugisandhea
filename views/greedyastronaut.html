Greedy Astronaut Game

Game Developer

2024

This application is made using Unity for my Mobile Programming midterms project. I made this 2D platformer game as a maze that a little astronaut has to go through while collecting as much coin as possible, and also go through a maze in a certain period of time. The door to the next level doesn't open as long as all the coins isn't collected yet, hence the name of the game, Greedy Astronaut.
