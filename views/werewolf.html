Office Werewolf Game

Game Developer

2024

This program is my Distributed Systems class final project in my third semester at college. This is a group project and I was assigned to be the game developer of the project. The project is about making a program to be used in an office's distributed systems with internal, guests, and server computers. The program is a werewolf game that can be played automatically without any assigned moderator both by officer employees' computers and guests' computers' terminals. I was responsible for making the full game logic, including the game rules, player roles, and game flow.
